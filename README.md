# ❤️ CardioCare – Cardiovascular Disease Prediction

A Machine Learning-powered web application that predicts the risk of cardiovascular disease based on patient health parameters. Built using Flask and Scikit-learn with a modern, responsive web interface.

## 🌐 Live Demo
🔗 https://yashwanthiga.pythonanywhere.com

## 🚀 Features
- ❤️ Predicts cardiovascular disease risk
- 📊 Generates a detailed patient report
- 🤖 Machine Learning (Random Forest Classifier)
- 📱 Responsive and user-friendly interface
- ⚡ Instant prediction results
- 🌍 Accessible through any web browser

## 🛠️ Tech Stack
- 🐍 Python
- 🌐 Flask
- 🤖 Scikit-learn
- 🔢 NumPy
- 💼 Joblib
- 🎨 HTML5
- 🎨 CSS3
- ⚡ JavaScript

## 📋 Input Parameters
- Age
- Gender
- Height
- Weight
- Systolic Blood Pressure
- Diastolic Blood Pressure
- Cholesterol Level
- Glucose Level
- Smoking Status
- Alcohol Consumption
- Physical Activity
